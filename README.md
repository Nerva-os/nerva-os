# Nerva OS

**Your AI for everything that matters.**

## What it is

Nerva OS turns Claude Code into a fully-trained assistant for your business AND personal life that lives on your computer. It knows your business, remembers what you tell it, and runs 37 specialist skills covering sales, marketing, operations, finance, and admin. One install. No subscriptions to wire up. No technical setup.

## What's inside

- **37 specialist skills** — sales, marketing, finance, ops, admin, research, and more
- **Safety layer** — a constitution-style guardrail system that keeps the assistant inside the boundaries you set
- **Workspace template** — a pre-built `my-assistant/` folder with memory, inbox, and the brain file already wired up
- **Tutorials & install prompts** — paste one prompt into Claude Code and your assistant builds itself
- **Engineering Pack (paid upsell)** — 56 additional skills for technical work, deployment, and code review

## Who it's for

Non-technical small business owners who want an AI assistant that actually understands their business — not a generic chatbot. If you can paste a prompt into a window, you can install this.

## How to install

1. Buy the kit, clone the repo, and open Claude Code.
2. Open [`SETUP-PROMPT.md`](SETUP-PROMPT.md) and paste the prompt inside into Claude Code.
3. Claude does the rest.

Prefer to install manually? See [`INSTALL.md`](INSTALL.md) for a step-by-step plain-English guide.

For a tour of what's actually in the kit before you start, see [`WELCOME.md`](WELCOME.md).
